# Daily Briefing Script

A simple Python application that provides a quick weather report and top news headlines for a topic of your choice.

## Prerequisites

* **Python 3.x** installed on your system.
* **API Keys** for [NewsAPI](https://newsapi.org/) and [OpenWeatherMap](https://openweathermap.org/api).


## How to run this project:
*Clone this repository.
*Create a virtual environment and install requirements: pip install -r requirements.txt
*Create a file named .env in the project root.
*Copy the contents of .env.example into your new .env file.
*Replace the placeholders with your own NewsAPI and OpenWeatherMap API keys.
*Run the script: python main.py
